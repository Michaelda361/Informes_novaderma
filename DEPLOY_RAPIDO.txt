  GUÍA RÁPIDA: DESPLEGAR EN RENDER.COM (5 MINUTOS)
═══════════════════════════════════════════════════════════════

PASO 1: SUBIR A GITHUB
-----------------------
1. Ve a https://github.com/new
2. Crea un repositorio nuevo (ej: "informes-novaderma")
3. NO marques "Add README" ni ".gitignore"
4. Click en "Create repository"

5. En tu computadora, abre la terminal en la carpeta del proyecto:
   
   git init
   git add .
   git commit -m "Primer commit"
   git branch -M main
   git remote add origin https://github.com/TU_USUARIO/informes-novaderma.git
   git push -u origin main


PASO 2: DESPLEGAR EN RENDER
----------------------------
1. Ve a https://render.com
2. Click en "Get Started" o "Sign Up"
3. Regístrate con tu cuenta de GitHub

4. En el dashboard de Render:
   - Click en "New +" (arriba derecha)
   - Selecciona "Web Service"
   
5. Conecta tu repositorio:
   - Busca "informes-novaderma"
   - Click en "Connect"

6. Configuración (Render lo detecta automáticamente):
   - Name: informes-novaderma
   - Environment: Python 3
   - Build Command: pip install -r requirements.txt
   - Start Command: gunicorn app:app
   - Plan: FREE
   
7. Click en "Create Web Service"

8. ESPERA 5-10 MINUTOS mientras se despliega

9. ¡LISTO! Tu app estará en:
   https://informes-novaderma.onrender.com


NOTAS IMPORTANTES:
------------------
✅ Es completamente GRATIS
✅ Se actualiza automáticamente cuando haces push a GitHub
✅ Incluye HTTPS (seguro)
⚠️  Se "duerme" después de 15 min sin uso (tarda 30 seg en despertar)


ACTUALIZAR LA APP:
------------------
Cuando hagas cambios:

git add .
git commit -m "Descripción del cambio"
git push

Render detectará el cambio y desplegará automáticamente.


ALTERNATIVA MÁS RÁPIDA: RAILWAY.APP
------------------------------------
1. Ve a https://railway.app
2. Regístrate con GitHub
3. Click en "New Project" → "Deploy from GitHub repo"
4. Selecciona tu repositorio
5. Railway lo despliega automáticamente
6. Ve a Settings → Generate Domain
7. ¡Listo!

Railway da $5 gratis al mes (suficiente para uso moderado)


¿PROBLEMAS?
-----------
- Revisa los logs en el dashboard de Render
- Verifica que todos los archivos estén en GitHub
- Asegúrate de que requirements.txt esté correcto
