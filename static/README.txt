INSTRUCCIONES PARA EL LOGO
==========================

Para que el logo aparezca en los PDFs:

1. Coloca el archivo del logo en esta carpeta (static/)
2. Renómbralo como: logo.png
3. Formato recomendado: PNG con fondo transparente
4. Tamaño recomendado: 200x50 píxeles (máximo altura 50px)

Si no colocas un logo, aparecerá el nombre de la empresa en texto.
